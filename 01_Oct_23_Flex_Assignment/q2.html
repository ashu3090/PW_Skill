Role of Flex Layout Properties: 1 flex-direction: Set the direction of the flexible items inside the
element in reverse order: div { display: flex; flex-direction: row-reverse; } 2. flex-wrap: The flex-wrap property specifies whether the flexible items should wrap or not. #main { width: 200px; height: 150px; border: 1px solid #c3c3c3; display: flex; flex-wrap: wrap; } 3. gap: The gap property defines the size of the gap between the rows and between the columns in flexbox. 4.align-item: The align-items property specifies the default alignment for items inside a flexbox or grid container. In a flexbox container, the flexbox items are aligned on the cross axis, which is vertical by default 5. justify-content: The justify-content property aligns the flexible container's items when the items do not use all available space on the main-axis (horizontally).
